6. DKIM Recommended

ProtonMail supports DKIM signing for custom domains. To use DKIM authentication, please add the following CNAME record into your DNS settings for this domain. This can typically be done in the control panel of your domain name registrar.

Please add all 3 of the following CNAME records. Note, DNS records can take several hours to update.

Learn more

Type Host name Value / Data / Points to
CNAME {{ config.Hostname }} {{ config.CNAME }}

IMPORTANT: Some registrars do not accept CNAME values with a period at the end (while others require it). If your registrar does not accept your CNAME records, please delete the period at the end of each CNAME value and try again.

Keep those records in your DNS for as long as you want to use DKIM. You can change its Value to off to disable DKIM
